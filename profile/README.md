# KUTIAN VT

KUTIAN Microelectronics focuses on wireless communication, video transmission, and control-link products for unmanned systems.

We provide high-integration video/data transmission modules, link boxes, remote controllers, and supporting development resources for UAVs, ground robots, inspection platforms, security systems, and other industrial applications.

This GitHub organization hosts product documentation, SDK references, examples, and integration resources for KUTIAN video transmission products.

## Product Lines

- **P401 Series Wireless Modules**  
  High-bandwidth wireless data transmission modules for embedded integration.

- **L4 / L4-MINI Video Transmission Boxes**  
  Ready-to-use video/data transmission devices for UAV and robotics applications.

- **K4R-MINI Remote Controller**  
  Integrated remote control system with video transmission, telemetry, and ground station support.

# KUTIAN-VT

酷天微电子专注于无人系统无线通信、图像传输和控制链路产品，面向无人机、无人车、无人船、机器人、安防巡检和行业应用客户，提供无线数传模组、图传链路、遥控终端及配套技术支持。

KUTIAN Microelectronics develops wireless communication, video transmission, and control-link products for unmanned systems and industrial remote-control applications.

This GitHub organization hosts public product documents, SDK references, integration notes, and example resources for KUTIAN video/data transmission products.

## Product Lines

| Product | Description | Key Features |
|---|---|---|
| P401 / P410 / P451 Wireless Modules | High-bandwidth 2x2 MIMO wireless data transmission modules for embedded integration | TDD, OFDM, adaptive frequency hopping, dynamic MCS, configurable 1.25-40 MHz bandwidth |
| L4 Link Suite | Ground-side AP + air-side DEV video/data link for unmanned platforms | Ethernet, USB, UART, and SBUS passthrough; low-latency link; 1.4 GHz / 2.4 GHz / 5.8 GHz options |
| L4-MINI | Compact P401-based video transmission board | 36 x 36 mm board, 30.5 x 30.5 mm mounting holes, MMCX/IPEX antenna options |
| K4R-MINI Remote Controller | Integrated remote-control and ground-station terminal | Android system, high-brightness display, SBUS/MAVLink/network transmission, long-range link options |

## Application Areas

- UAV video transmission, telemetry, and remote control
- UGV / USV communication and command links
- Payload, gimbal, IP camera, and LiDAR data backhaul
- Long-range Ethernet, UART, USB, and SBUS transparent transmission
- Industrial inspection, security monitoring, and special-purpose wireless links
